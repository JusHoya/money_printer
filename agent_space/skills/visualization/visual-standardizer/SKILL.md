# Visual Standardizer

You are the guardian of the "Hoya Box" visual identity. Your role is to ensure every pixel generated by the agent system—whether a scientific plot or a marketing asset—adheres to a strict, high-contrast, "Space-Cyberpunk" aesthetic.

## When to use this skill
*   **Creating Plots:** When writing Python code (Matplotlib, Plotly) to visualize data.
*   **Generating Images:** When crafting prompts for image generation tools.
*   **UI Mockups:** When designing HTML/CSS interfaces.
*   **Formatting:** When styling Markdown tables or documents for visual impact.

## The "Hoya Box" Aesthetic
*   **Theme:** Deep Space / Cyberpunk / Glassmorphism.
*   **Backgrounds:** Deep dark blues/blacks (`#0B0C10`, `#1F2833`). Avoid pure black.
*   **Accents:** Neon Cyan (`#66FCF1`), Neon Pink (`#FF007F`), Electric Purple (`#BC13FE`).
*   **Typography:** Monospace for data, clean sans-serif (Inter/Roboto) for headers.

## Instructions

### 1. Scientific Plotting (Python)

**Mandatory:** Always apply these style parameters when generating plots.

#### A. Plotly (Preferred for Interactivity)
Use this template setup:

```python
import plotly.graph_objects as go
import plotly.io as pio

# Hoya Box Theme Definition
hoya_template = go.layout.Template()
hoya_template.layout = dict(
    paper_bgcolor='#0B0C10',
    plot_bgcolor='#1F2833',
    font=dict(family='Courier New, monospace', color='#C5C6C7'),
    title=dict(font=dict(color='#66FCF1', size=24)),
    xaxis=dict(gridcolor='#45A29E', zerolinecolor='#66FCF1'),
    yaxis=dict(gridcolor='#45A29E', zerolinecolor='#66FCF1'),
    colorway=['#66FCF1', '#45A29E', '#C5C6C7', '#FFFF00', '#FF007F'] # Neon palette
)
pio.templates['hoya_dark'] = hoya_template
pio.templates.default = 'hoya_dark'
```

#### B. Matplotlib (Static)
Use this configuration:

```python
import matplotlib.pyplot as plt

plt.style.use('dark_background')
plt.rcParams.update({
    'figure.facecolor': '#0B0C10',
    'axes.facecolor': '#1F2833',
    'axes.edgecolor': '#45A29E',
    'axes.labelcolor': '#66FCF1',
    'text.color': '#C5C6C7',
    'xtick.color': '#45A29E',
    'ytick.color': '#45A29E',
    'grid.color': '#45A29E',
    'grid.alpha': 0.3,
    'lines.linewidth': 2,
    'font.family': 'monospace'
})
# Primary Colors: Cyan (#66FCF1), Grey (#C5C6C7), Teal (#45A29E)
```

### 2. Image Generation Prompts

When using `generate_image`, append this style string to the specific subject prompt:

> "... in a high-fidelity cinematic style, deep space atmosphere, neon cyan and electric purple lighting, dark obsidian background, glassmorphism elements, sharp focus, 8k resolution, futuristic UI overlay."

### 3. UI & Web Design

*   **Glassmorphism:** Use semi-transparent backgrounds with blur.
    *   `background: rgba(11, 12, 16, 0.7); backdrop-filter: blur(10px);`
*   **Borders:** Thin, glowing borders.
    *   `border: 1px solid rgba(102, 252, 241, 0.3);`
    *   `box-shadow: 0 0 15px rgba(102, 252, 241, 0.2);`

## Best Practices
*   **Contrast is King:** Make sure data lines pop against the dark background.
*   **Less is More:** Use negative space. Don't clutter the "Deep Space" vibe.
*   **Consistency:** Never mix "standard default white" plots with Hoya Box styles.
