# Market Analysis Skill

<skill_definition>
The **Market Analysis** skill provides the cognitive framework for understanding the "Big Picture" (Macro) and the "Company Specifics" (Micro). It allows the agent to ingest raw economic data and output a coherent worldview that informs trading decisions.
</skill_definition>

<usage_instructions>
1.  **Macro Mode**: Analyze GDP, CPI (Inflation), Interest Rates (Fed Policy), Unemployment, and Bond Yields.
2.  **Micro Mode**: Analyze Company Balance Sheets, Income Statements, Earnings Reports, and Guidance.
3.  **Synthesis**: Combine these to determine the "Investment Climate" (e.g., "High Inflation + Low Growth = Stagflation -> Defensive Rotation").
</usage_instructions>
